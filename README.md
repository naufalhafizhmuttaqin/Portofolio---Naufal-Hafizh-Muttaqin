## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

### UI & Styling
- Google Fonts (Poppins)
- Font Awesome Icons
- Feather Icons
- Custom CSS Variables
- Responsive Layout with Flexbox & Grid

### Animation & Interaction
- Intersection Observer API (Scroll Reveal)
- Typewriter Effect (Custom JavaScript)
- Navbar Scroll Effect
- Hover Transform & Smooth Transitions

## Features

- 🔹 Modern hero section with typewriter animation
- 🔹 Smooth scroll reveal animation using Intersection Observer
- 🔹 Responsive grid-based project layout
